# Phishing Attack Project

This project demonstrates a simulated phishing attack using the tool ZPhisher. The goal is to educate users about phishing techniques and promote cybersecurity awareness in an ethical and controlled environment.

## Features
- **Phishing Page Creation**: Generates a fake Instagram login page to simulate phishing.
- **Email Phishing**: Crafts a phishing email with a hidden link to the phishing page.
- **Analysis**: Observes user interactions to highlight vulnerabilities.
- **Preventive Measures**: Provides tips to recognize and avoid phishing attacks.

## Tools Used
- **ZPhisher**: Open-source phishing tool to replicate legitimate login pages.
- **Email Client**: Used to create and send phishing emails.

## Project Structure
```
├── ZPhisher_Setup      # Setting up the phishing tool
├── Phishing_Page       # Contains the generated phishing page
├── Email_Simulation    # Scripts and examples for crafting phishing emails
├── Awareness_Content   # Tips and preventive measures for end users
└── README.md           # Project documentation
```

## Ethical Guidelines
- **Educational Use Only**: This project is strictly for educational purposes.
- **Consent Required**: Any demonstration must have the informed consent of participants.
- **No Harm Policy**: No real credentials are collected or misused.

## How It Works
1. **Setup**: Install ZPhisher and generate a phishing page for Instagram.
2. **Email Crafting**: Create a phishing email embedding the phishing link.
3. **Simulation**: Share the email in a controlled environment.
4. **Awareness**: Analyze responses and educate users about phishing.

## Preventive Measures
- Verify URLs before entering credentials.
- Avoid clicking suspicious links in emails.
- Enable two-factor authentication (2FA).
- Be cautious of emails with urgent requests.
